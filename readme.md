## Knight Encrypter

> **`Encrypt`** and **`Decrypt`** any file from the command line.

* `Fast` **`60 MB/s encryption`** **`45 MB/s decryption`**
* `Password protect files`
* `Hard to break encryption`

`How to use this app? Enter:`
```sh 
knightencrypter.py help
```
